기능 목록
1. 입력
- 매칭, 조회, 초기화, 종료 입력
  - 예외 -> 해당하는 문자가 아닌 경우
- 과정, 레벨, 미션 입력받기
  - 예외  
    1. , 폼으로 입력을 받지 않은 경우
    2. 과정이 존재하지 않는 경우 
    3. 레벨이 존재하지 않는 경우
    4. 미션이 존재하지 않는 경우
2. 페어 매칭
- 해당 페어 매칭 정보가 존재하는지 확인하기
- 매칭 정보가 없는 경우
    - 매칭 -> 셔플 이용
- 매칭된 정보가 있는 경우 -> 메세지 출력
3. 페어 조회
- 입력받은 정보를 통해 페어 정보를 조회한다.
4. 페어 초기화
- 모든 페어를 초기화한다.
5. 종료
